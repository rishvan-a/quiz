Networking Roadmap

Click topics to mark complete. Progress saves automatically.

Phase 1 — Fundamentals

1. What is a Network?

Basic

LAN, WAN, MAN, PAN — types of networks, their scope, and real-world examples.

2. OSI Model (7 Layers)

Important

Physical, Data Link, Network, Transport, Session, Presentation, Application — each layer's function, protocols, and devices.

3. TCP/IP Model & Protocol Suite

Important

4-layer model vs OSI, TCP vs UDP, 3-way handshake, ports & sockets, IP addressing fundamentals.

4. IP Addressing (IPv4 & IPv6)

Important

Classes (A/B/C/D/E), private vs public IP, subnet masks, CIDR notation, IPv6 format.

5. Subnetting & CIDR

Critical

Subnetting formulas, VLSM, supernetting, practice problems. Essential for network design.

6. MAC Addresses & ARP

Basic

MAC address format, ARP protocol, ARP table, gratuitous ARP, ARP poisoning intro.

7. DNS (Domain Name System)

Important

DNS resolution, record types (A, AAAA, CNAME, MX, TXT, NS), DNS caching, DNS attacks.

8. DHCP & NAT

Basic

DORA process, DHCP relay, static vs dynamic NAT, PAT, how NAT protects internal networks.

Phase 2 — Intermediate

9. Routing Protocols

Important

Static routes, RIPv2, OSPF, EIGRP fundamentals, administrative distance, metrics.

10. Switching & VLANs

Important

MAC table, STP, VLAN config, trunk ports, inter-VLAN routing, VTP.

11. Firewalls & ACLs

Critical

Standard vs extended ACLs, stateful vs stateless, firewall zones, DMZ setup.

12. VPN Technologies

Important

IPSec (AH/ESP), SSL/TLS VPN, site-to-site vs remote access, tunnel modes, IKE phases.

13. Wireshark & Packet Analysis

Critical

Capture filters, display filters, analyzing TCP handshakes, DNS queries, HTTP traffic.

14. Network Troubleshooting Tools

Basic

ping, traceroute, nslookup, dig, netstat, ss, nmap (basic), tcpdump mastery.

Phase 3 — Advanced

15. BGP & MPLS

Advanced

BGP attributes, AS numbers, route manipulation, MPLS labels, LSP, VPN over MPLS.

16. SDN & Network Automation

Advanced

OpenFlow, controllers, Python networking (Netmiko, NAPALM), Ansible for networks.

17. Network Security Architecture

Advanced

Zero Trust, micro-segmentation, 802.1X NAC, IDS/IPS placement, honeynets.

18. IPv6 Deep Dive

Important

IPv6 addressing, SLAAC, DHCPv6, security considerations, tunneling, migration.

Cybersecurity Roadmap

Core security concepts every professional must know.

Phase 1 — Fundamentals

1. CIA Triad & Security Principles

Critical

Confidentiality, Integrity, Availability, non-repudiation, authentication, authorization.

2. Types of Cyber Threats

Important

Malware, ransomware, phishing, social engineering, DDoS, MITM, zero-day attacks.

3. Cryptography Basics

Critical

Symmetric (AES, DES) vs Asymmetric (RSA, ECC), hashing (SHA, MD5), digital signatures, PKI.

4. Access Control Models

Important

MAC, DAC, RBAC, ABAC, identity management, MFA, SSO, least privilege.

5. Linux for Security

Critical

File permissions, process management, bash scripting, log files, user/group management.

6. Security Frameworks & Compliance

Important

NIST CSF, ISO 27001, CIS Controls, GDPR, HIPAA, PCI-DSS.

Phase 2 — Detection & Defense

7. SIEM & Log Analysis

Critical

Splunk, ELK Stack, log aggregation, correlation rules, dashboards, alerting.

8. IDS/IPS

Important

Snort, Suricata, signature vs anomaly detection, rule writing, deployment.

9. Vulnerability Assessment

Important

Nessus, OpenVAS, CVSS scoring, false positives, remediation prioritization.

10. Incident Response

Critical

NIST IR: Preparation → Detection → Containment → Eradication → Recovery.

11. OWASP Top 10

Critical

SQL Injection, XSS, CSRF, Broken Auth, Security Misconfiguration — mitigations.

Phase 3 — Advanced

12. APTs & MITRE ATT&CK

Advanced

APT lifecycle, nation-state actors, kill chain analysis, MITRE framework mapping.

13. Cloud Security

Advanced

AWS/Azure/GCP security, IAM, S3 security, VPC, shared responsibility model.

14. Digital Forensics

Advanced

Disk/memory forensics, Volatility, network forensics, chain of custody, FTK/Autopsy.

15. Malware Analysis

Advanced

Static/dynamic analysis, Cuckoo sandbox, reverse engineering with Ghidra/IDA.

Ethical Hacking Roadmap

Offensive security — think like an attacker to defend better.

Disclaimer: All techniques are for educational purposes and authorized testing only. Unauthorized access is illegal.

Phase 1 — Recon & Scanning

1. Setting Up Your Lab

Critical

Kali Linux, VirtualBox, vulnerable VMs (Metasploitable, DVWA), lab environment setup.

2. Passive Reconnaissance (OSINT)

Important

WHOIS, Shodan, Google Dorking, theHarvester, Maltego — gathering info without touching target.

3. Active Scanning with Nmap

Critical

Scan types (SYN, UDP, ACK), service/version detection, OS detection, NSE scripts.

4. Enumeration & Fingerprinting

Important

SMB, LDAP, DNS enumeration, SNMP, Gobuster/Dirb fuzzing, banner grabbing.

5. Vulnerability Scanning

Important

Nessus, OpenVAS, Nikto — automated scanning, interpreting results, manual verification.

Phase 2 — Exploitation

6. Metasploit Framework

Critical

Msfconsole, exploits, payloads (reverse shells), meterpreter, post-exploitation modules.

7. Web Application Hacking

Critical

SQL Injection, XSS, CSRF, file upload bypass, Burp Suite workflow, SQLMap.

8. Password Attacks

Important

Dictionary/brute force, rainbow tables, John the Ripper, Hashcat, Hydra, password spraying.

9. Network Attacks

Critical

ARP spoofing, DNS spoofing, MITM with Ettercap, SSL stripping, session hijacking.

10. Privilege Escalation

Critical

Linux SUID/cron/kernel exploits, Windows token impersonation, unquoted paths, LinPEAS/WinPEAS.

Phase 3 — Post-Exploitation & Advanced

11. Post-Exploitation & Persistence

Advanced

Backdoors, rootkits, pivot/tunneling, lateral movement, data exfiltration.

12. Active Directory Attacks

Advanced

Kerberoasting, AS-REP Roasting, Pass-the-Hash/Ticket, Golden/Silver Tickets, BloodHound.

13. Reverse Engineering & Exploit Dev

Advanced

Buffer overflows, shellcode, ROP, Ghidra/IDA analysis.

14. Report Writing & Pro Pentesting

Important

Professional reports, risk ratings, executive summaries, PTES methodology.
